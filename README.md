# KelvinFL

Адаптивный HTML/CSS-лендинг-портфолио. Проект сделан как статическая веб-страница с hero-блоком, секциями услуг/работ и простой интерактивностью на JavaScript.

## Что внутри

- Главная страница `index.html`.
- Адаптивная вёрстка под разные экраны.
- Стили, разделённые на базовые и media-запросы.
- Изображения и SVG-иконки для интерфейса.
- Небольшой JavaScript для поведения страницы.

## Стек

- HTML5
- CSS3
- JavaScript
- Responsive layout

## Структура

```text
.
├── index.html
├── css/
│   ├── main.css
│   ├── media.css
│   └── reset.css
├── js/
│   └── main.js
└── img/
```

## Запуск

Откройте `index.html` в браузере или запустите локальный статический сервер:

```bash
python -m http.server 8000
```

После запуска страница будет доступна по адресу:

```text
http://127.0.0.1:8000
```

## Статус

Учебный frontend-проект для практики адаптивной вёрстки и базовой структуры лендинга.
